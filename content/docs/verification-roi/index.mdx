## Why Verification Matters

Detection and action are not enough.

You must confirm:

- Did the action succeed?  
- Did the issue resolve?  

---

## Verification Process

After execution:

- Query external systems  
- Confirm state change  
- Validate outcome  

---

## Value Types

### Recovered Revenue

Directly regained revenue.

### Avoided Loss

Prevented revenue loss.

---

## ROI Tracking

Solvren tracks:

- Total recovered revenue  
- Total avoided loss  
- Value per playbook  
- Value over time  

---

## Confidence

Each value event includes:

- Amount  
- Confidence score  

---

## Outcome

A provable, auditable record of value generated by the system.
