## Overview

The ROI dashboard surfaces verified value to executives. It shows recovered revenue, avoided loss, and trends over time.

---

## Metrics

- **Total recovered** — Revenue restored by actions
- **Total avoided** — Loss prevented
- **By playbook** — Value attributed per playbook
- **By period** — Trends over time

---

## Use Cases

- Prove platform value to leadership
- Compare playbook performance
- Track time-to-value and ramp
