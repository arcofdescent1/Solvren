## Change Cannot Be Submitted

This usually occurs when required information is missing.

Check:

• evidence checklist completion  
• required fields in intake  
• approval mappings

---

## Missing Approvers

If no approvers appear, verify:

• approval role configuration  
• user role assignments  
• approval mapping settings

Administrators can configure these in the organization settings.

---

## Notifications Not Received

Check the following:

• user notification preferences  
• Slack integration configuration  
• organization email settings

---

## Evidence Not Accepted

Some organizations require specific evidence types.

Ensure that required evidence items are completed before submitting the change.

---

## Unable to Find Historical Changes

Use search filters such as:

• domain  
• system  
• reviewer  

Search supports full-text queries across change titles and descriptions.
