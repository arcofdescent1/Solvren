## Overview

**First value** is the moment you see recovered revenue or avoided loss attributed to a Solvren action. This page summarizes the path.

---

## Steps

1. **Connect** — Integrate at least one revenue system
2. **Detect** — Enable detector packs; see first issue
3. **Enable** — Turn on a playbook for that issue type
4. **Verify** — Confirm action succeeded; see value in ROI dashboard

See [Get Started](/docs/get-started) for detailed guides.

---

## Time to First Value

With healthy integrations and configured detectors, first value can typically be achieved within days of setup.
