## Overview

Data flows from external systems through integrations into signals, then through detectors to issues, and finally through actions to verification and ROI.

---

## Flow Summary

1. **Ingestion** — Integrations pull or receive data → signals
2. **Detection** — Detectors analyze signals → issues
3. **Prioritization** — Impact engine scores issues
4. **Action** — Playbooks execute (recommend, simulate, or write-back)
5. **Verification** — Outcomes checked → value attributed
6. **Reporting** — ROI dashboard and metrics updated
