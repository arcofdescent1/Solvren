## Platform Layers

- **Integrations** — Connect to revenue systems
- **Identity graph** — Entity resolution across systems
- **Signals** — Ingested data streams
- **Detectors** — Issue detection logic
- **Impact engine** — Revenue at risk quantification
- **Execution / write-back** — Action execution
- **Verification / ROI** — Value attribution
- **Safe automation** — Policies and approvals
- **Onboarding / performance** — Getting to value and playbook metrics
