## Overview

The final step in the value loop is **verification**. Solvren tracks which actions succeeded and quantifies the **recovered revenue** or **avoided loss** they produced.

---

## What You'll Do

1. Open the Verification or ROI area in the dashboard
2. Find issues where actions have been executed and verified
3. Review recovered vs avoided value
4. Use the ROI dashboard for executive visibility

---

## Recovered vs Avoided Value

- **Recovered revenue** — Value that was lost (e.g., churn, billing error) and restored by an action
- **Avoided loss** — Value that would have been lost if the issue had not been addressed

Both contribute to the total ROI Solvren proves.

- A connected system
- A detected issue
- An enabled playbook
- Verified value
